# AreeqAttire Shopify Assets

This repository holds the raw assets and source codes for AreeqAttire's Shopify theme modifications, including the 3D spinning logo animation bundle.

The site is hosted and managed completely on Shopify using the Horizon theme.

## Directories
- `shopify-bundle/`: Contains the build files for the 3D visual assets, including `hero-3d.liquid` and `areeq-hero.iife.js`.
